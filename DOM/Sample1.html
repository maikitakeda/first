Sample1

このサンプルでは、ボタンがクリックされた時に、<p></p>の要素を変更します。
ボタンをクリックするとchange()メソッドを呼び出し、<p></p>の要素を変えます。

変わる前



Sample1.html
<p id="before">変わる前</p>
<input type="button" value="クリックすると変わります" onClick="change()">

<script type="text/javascript">
function change() {
var p=document.getElementById("before");
p.innerHTML="変わりました";
}
</script>
