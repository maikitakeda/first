このサンプルでは、ボタンがクリックされた時に、<p></p>文字色を変更します。
ボタンをクリックするとchange()メソッドを呼び出し、<p></p>の文字色を変えます。

黒から赤へ



Sample2.html
<p id="before">黒から赤へ</p>
<input type="button" value="クリックすると変わります" onClick="change()">

<script type="text/javascript">
function change() {
var p=document.getElementById("before");
p.style.color="red";
}
</script>
