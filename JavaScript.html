JavaScript

JavaScriptを用いることで、Webページを動的に操作することができます。
例えば、あるセレクトリストの値を変えたら、それに伴って他のセレクトリストの値が変わるなどです。

コピー制御

たとえば、Body要素のコピーを防ぐ場合は以下のように記述します。

<body oncopy="alert('コピーしないでね！！'); return false;">

読み込み時に処理を行う

例えば、読み込む際にリンク先を指定する際は以下のように記述します。
読み込む際にinitメソッドを実行します。
initメソッドでは、idがTopPageである要素を取得し、リンク先を指定しています。

onload = init;

function init(){
    var link = document.getElementById("TopPage");
    link.href="http://maikitakeda.github.io/first/Index.html";
}
