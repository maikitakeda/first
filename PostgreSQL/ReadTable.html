検索

SQLをマスターするためには、検索が非常に重要です。
効率の良い検索を行えると、その分性能の向上につなげることができます。

全件検索
取得する列を指定する
取得する条件を指定する
内部結合して検索する

全件検索

全件検索する場合は下記のように記述します。
下記の例では、顧客テーブルから全ての値を取得します。
SELECT * FROM customer;

取得する列を指定する

取得する列を指定する場合は下記のように記述します。
下記の例では、顧客テーブルから名前のみを取得しています。
SELECT name FROM customer;

取得する条件を指定する

取得する条件を指定する場合は下記のように記述します。
下記の例では、顧客テーブルからcustomer_idが1の顧客の情報を取得しています。
SELECT * FROM customer WHERE customer_id = 1;

内部結合して検索する

二つのテーブルを結合して、値を取得したい場合は下記のように記述します。
下記の例では、顧客テーブルと購入履歴テーブルをcustomer_idで内部結合し、
顧客名と金額を取得しています。
「INNER JOIN 結合したいテーブル名 ON 結合条件」という書き方をします。
また、テーブル名の後に名前をつけると、列名を指定する時にいちいち長いテーブル名を書く必要がなくなりますよ。
SELECT c.customer_name, b.sum FROM customer c INNER JOIN bought_history b ON
c.customer_id = b.customer_id;
