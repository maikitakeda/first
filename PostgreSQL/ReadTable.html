検索

SQLをマスターするためには、検索が非常に重要です。
効率の良い検索を行えると、その分性能の向上につなげることができます。

全件検索
取得する列を指定する 取得する条件を指定する

全件検索

全件検索する場合は下記のように記述します。
下記の例では、顧客テーブルから全ての値を取得します。
SELECT * FROM customer;

取得する列を指定する

取得する列を指定する場合は下記のように記述します。
下記の例では、顧客テーブルから名前のみを取得しています。
SELECT name FROM customer;

取得する条件を指定する

取得する条件を指定する場合は下記のように記述します。
下記の例では、顧客テーブルからidが1の顧客の情報を取得しています。
SELECT * FROM customer WHERE id = 1;
