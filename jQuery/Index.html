jQueryは軽量で高速なJavaScriptライブラリです。
簡単なサンプルを用意しているので試してみてください。

フォームの値を取得する
テキストの値を取得する
1個のクラスから複数の値を取得し、配列にする
条件に合致する値をハイライト表示する

フォームの値を取得する

たとえば、classNameクラスのinput要素から値を取得する場合は以下のように記述します。

$('.className').val();

テキストの値を取得する

たとえば、classNameクラスのtext要素から値を取得する場合は以下のように記述します。

$('.className').text();

1個のクラスから複数の値を取得し、配列にする

たとえば、classNameクラスのtext要素を配列として取得する場合は以下のように記述します。
map()でreturnする値を配列に加えます。

var array = $('.className').map(function(){
    return $(this).text();
}).get();

条件に合致する値をハイライト表示する

たとえば、入力した条件がinput_phone_numberクラスに、検索結果がphone_numberクラスに属するときに
入力条件が検索結果に合致する場合のみハイライト表示するには、以下のように記述します。

var inputPhoneNumber = "";
inputPhoneNumber = $('.input_phone_number').val();

if (inputPhoneNumber != "") {
    $('.phone_number1').each(function (i, e) {
        if ($(e).text() == inputPhoneNumber) {
            $(e).css("background-color", "#00ffff");
        }
    });
});
